Win32 Installation:
	1. Install QMM
	2. Make a qmmaddons/advchat/dlls directory inside your mod directory and place advchat_qmm.dll here
	3. Add "qmmaddons/advchat/dlls/advchat_qmm.dll" as an entry in the plugins list in qmm.ini

Linux Installation:
	1. Install QMM
	2. Make a qmmaddons/advchat/dlls directory inside your mod directory and place advchat_qmm.so here
	3. Add "qmmaddons/advchat/dlls/advchat_qmm.so" as an entry in the plugins list in qmm.ini
